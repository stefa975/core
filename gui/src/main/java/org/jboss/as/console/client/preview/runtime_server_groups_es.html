Grupos de servidores

Un grupo de servidores es un conjunto de instancias de servidor que se configuran y se administran como una sola. En un dominio administrado, cada instancia del servidor de aplicaciones es un miembro de un grupo de servidores, incluso si hay un solo servidor en un grupo de servidores.

Es la responsabilidad del controlador de dominios y de los controladores de hosts asegurarse de que todos los servidores de un grupo de servidores tengan una configuración consistente. Todas las instancias del servidor usan el mismo perfil y tienen el mismo contenido de implementación.
