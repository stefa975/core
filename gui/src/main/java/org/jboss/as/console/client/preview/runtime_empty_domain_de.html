Laufzeit

Hosts, Servergruppen und Server verwalten. Laufzeitdienste, wie Protokolldateien, JVM-Metriken und untersystemspezifische Laufzeitdaten, anzeigen und überwachen.

Allgemeine Konfigurationsaufgaben

  • Servergruppen erstellen und konfigurieren.
  • Server erstellen und zu Servergruppen zuweisen.
  • Metriken anzeigen und überwachen.
  • Serverstatus untersuchen.
  • Anwendungsbezogene Laufzeitdienstnamen suchen.
