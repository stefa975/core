Pfade

Ein Pfad ist ein logischer Name für einen Dateisystempfad. Andere Abschnitte der Konfiguration können dann anhand des logischen Namens auf diese Pfade verweisen, anstatt die vollständigen Pfade mit einschließen zu müssen (die auf verschiedenen Rechnern variieren können). Es gibt mehrere vordefinierte Pfade, die von der Standardkonfiguration verwendet werden, beispielsweise der jboss.server.log.dir -Pfad, der das Verzeichnis für das Serverprotokoll definiert.
