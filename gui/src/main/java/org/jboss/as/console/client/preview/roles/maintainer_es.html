Mantenedor

La función del mantenedor tiene acceso para ver y modificar el estado del tiempo de ejecución y toda la configuración, excepto operaciones y datos confidenciales. La función del mantenedor es la función de uso general que no tiene acceso a operaciones ni datos confidenciales. La función del mantenedor permite que los usuarios reciban acceso casi completo para administrar el servidor sin darles a esos usuarios acceso a contraseñas y otra información confidencial.
