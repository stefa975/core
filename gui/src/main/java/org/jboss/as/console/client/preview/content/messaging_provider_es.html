Proveedor de mensajería

Un proveedor de mensajería es una instancia del servidor de ActiveMQ.

Colas/Temas

Cree colas y temas utilizados por el proveedor de mensajería.

Conexiones

Administre aceptadores, que definen de qué manera se pueden realizar las conexiones al proveedor de mensajería.

Cree puentes, que consumen mensajes desde una cola fuente y los envían a una dirección meta, normalmente en un servidor diferente.

Agrupamiento

Defina grupos de transmisión y detección, que controlan de qué manera este proveedor gestiona y distribuye los conectores.

Administre conexiones de clúster para cargar el balance entre los nodos de un clúster.

Configuración del proveedor

Configure los ajustes de este proveedor de mensajería.
