Membresía

Se considera que un usuario está asignado a una función si:

  1. El usuario:
    • está enumerado como un usuario que se debe incluir en la función; o
    • es un miembro de un grupo que está enumerado para que se incluya en la función.
  2. El usuario no:
    • está enumerado como un usuario que se debe excluir de la función; o
    • es un miembro de un grupo que está enumerado para que se excluya de la función.

Las exclusiones tienen prioridad sobre las inclusiones.
