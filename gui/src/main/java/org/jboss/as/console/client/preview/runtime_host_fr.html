Configurations d'hôte

Un contrôleur d'hôte est un hôte physique ou virtuel qui interagit avec le contrôleur de domaine central pour configurer et gérer les instances de serveurs d'applications exécutées sur son hôte.

Tâches de configuration courantes

  • Configuration des interfaces réseau, des paramètres de JVM et d'autres propriétés pour l'hôte en question.
  • Création et gestion des instances de serveur pour l'hôte en question.
  • Affichage des fichiers de journalisation des serveurs.
