Associação

Um usuário será considerado como atribuído a uma função se:

  1. O usuário:
    • estiver listado como um usuário a ser incluído na função ou
    • for um membro de um grupo listado para ser incluído na função.
  2. O usuário não:
    • estiver listado como um usuário a ser excluído da função,
    • for um membro de um grupo listado para ser excluído da função.

As exclusões têm precedência sobre as inclusões.
