メッセージングプロバイダー

メッセージングプロバイダーは Active MQ サーバーインスタンスです。

キュー / トピック

このメッセージングプロバイダーによって使用されるキューおよびトピックを作成します。

接続

メッセージングプロバイダーへの接続方法を定義するアクセプターを管理します。

ソースキューからメッセージを消費するブリッジを作成し、通常は異なるサーバーにある目的のアドレスへ転送します。

クラスタリング

このプロバイダーによってコネクターが処理および配布される方法を制御するブロードキャストおよびディスカバリーグループを定義します。

クラスターのノード間で負荷分散するためのクラスター接続を管理します。

プロバイダー設定

このメッセージングプロバイダーの設定を指定します。
