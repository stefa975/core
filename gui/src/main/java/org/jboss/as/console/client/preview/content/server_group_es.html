Grupos de servidores

Un grupo de servidores es un conjunto de instancias de servidor que se administran y se configuran como una. En un dominio administrado, cada instancia de servidor de la aplicación es un miembro de un grupo de servidores. (Incluso si el grupo solo tiene un servidor, este sigue siendo un miembro de un grupo). Es la responsabilidad del controlador de dominios y los controladores del host asegurarse de que todos los servidores de un grupo de servidores tengan una configuración consistente. Todos deben estar configurados con el mismo perfil y deben tener el mismo contenido de implementación.
