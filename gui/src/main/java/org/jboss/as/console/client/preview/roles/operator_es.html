Operador

El rol del Operador extiende el rol de Monitor al agregar la capacidad de modificar el estado de tiempo de ejecución del servidor. Esto significa que los operadores pueden recargar y apagar el servidor, así como pausar y reanudar los destinos JMS. El rol del operador es ideal para los usuarios responsables de los hosts físicos o virtuales del servidor de aplicaciones para que puedan asegurarse de poder apagar y reiniciar los servidores correctamente cuando sea necesario. Los operadores no pueden modificar la configuración del servidor ni acceder a operaciones y datos confidenciales.
