Administrador

La función del administrador tiene acceso libre a todos los recursos y operaciones del servidor excepto el sistema de registro de auditorías. La función del administrador tiene acceso a datos y operaciones confidenciales. Esta función también configura el sistema de control de acceso. La función del administrador solo se necesita cuando se manejan datos confidenciales o se configuran usuarios y funciones. Los administradores no pueden cambiar sus permisos a Auditor o Superusuario.
