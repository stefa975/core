Contrôle d'accès

Les définitions de contrôle d'accès attribuent ou limitent les accès aux opérations de gestion.

Tâches de configuration courantes

  • Ajouter des utilisateurs et des groupes.
  • Attribuer des rôles aux utilisateurs et aux groupes.
  • Définir des rôles personnalisés pour les hôtes ou les groupes de serveur.
