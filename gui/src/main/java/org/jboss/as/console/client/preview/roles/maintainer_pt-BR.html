Mantenedor

A função Mantenedor tem acesso para exibir e modificar o estado do tempo de execução e todas as configurações, exceto dados e operações confidenciais. A função Mantenedor é a função de uso geral que não tem acesso a dados e operações confidenciais. Ela permite que os usuários tenham acesso completo para administrar o servidor, sem conceder a esses usuários acesso a senhas e outras informações confidenciais.
