Monitor

Die Monitor-Rolle weist die geringsten Berechtigungen auf und kann lediglich die aktuelle Konfiguration und den Status des Servers anzeigen. Diese Rolle ist für Benutzer gedacht, die die Serverleistung verfolgen und melden müssen. Monitor-Benutzer können die Serverkonfiguration nicht ändern und haben keinen Zugriff auf sensible Daten oder Vorgänge.
