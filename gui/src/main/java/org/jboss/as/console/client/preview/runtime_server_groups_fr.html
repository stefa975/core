Groupes de serveurs

Un groupe de serveurs est un regroupement d'instances de serveurs qui sont gérés et configurés en un. Dans un domaine géré, chaque instance de serveur d'application appartient à un groupe de serveurs, même s'il en est le seul membre.

Il incombe au contrôleur de domaine et aux contrôleurs d'hôtes de s'assurer que tous les serveurs d'un groupe disposent d'une configuration homogène. Toutes les instances de serveur utilisent le même profil et disposent d'un contenu de déploiement identique.
