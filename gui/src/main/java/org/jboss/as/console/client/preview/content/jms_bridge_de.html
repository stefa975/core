JMS-Bridge

Eine JMS-Bridge konsumiert Nachrichten von einem Quell-JMS-Ziel und sendet diese an ein Ziel-JMS-Ziel (in der Regel auf einem anderen Server).

Die Bridge kann auch zum Bridging von Nachrichten von anderen Nicht-Artemis-JMS-Servern verwendet werden, vorausgesetzt, sie sind JMS 1.1-konform.
