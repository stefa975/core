Servergruppen

Eine Servergruppe ist eine Reihe von Serverinstanzen, die als eine Einheit konfiguriert und verwaltet werden. In einer verwalteten Domain gehört jede Applikationserverinstanz zu einer Servergruppe, selbst wenn diese Servergruppe nur einen einzelnen Server enthält.

Der Domain Controller und die Host Controller müssen sicherstellen, dass alle Server in einer Servergruppe eine konsistente Konfiguration aufweisen. Alle Serverinstanzen verwenden dasselbe Profil und denselben Deployment-Inhalt.
