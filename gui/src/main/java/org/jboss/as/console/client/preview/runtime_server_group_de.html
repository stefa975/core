Servergruppe

Eine Servergruppe ist eine Reihe von Serverinstanzen, die als eine Einheit konfiguriert und verwaltet werden. In einer verwalteten Domain gehören alle Serverinstanzen zu einer Servergruppe. Alle Serverinstanzen in einer Servergruppe verwenden dasselbe Profil und denselben Deployment-Inhalt.

Allgemeine Konfigurationsaufgaben

  • Neue Servergruppen hinzufügen.
  • Server erstellen und zu Servergruppen zuweisen.
  • Server starten, stoppen und erneut laden.
  • Serverprotokolldateien anzeigen.
