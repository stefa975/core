Fournisseur de messagerie

Un fournisseur de messagerie est une instance de serveur ActiveMQ.

Files d'attente / Rubriques

Création de files d'attente et de rubriques utilisées par ce fournisseur de messagerie.

Connexions

Gestion d'accepteurs, qui définissent la manière dont les connexions peuvent être effectuées au fournisseur de service.

Créations de ponts, qui consomment des messages à partir d'une file d'attente source et les transfère à une adresse cible, généralement sur un autre serveur.

Clustering

Définition des groupes de diffusion et de détection, qui contrôlent la manière dont les connecteurs sont gérés et distribués par ce fournisseur.

Gestion des connexions de cluster dans le but d'équilibrer les charges entre les noeuds d'un cluster.

Paramètres Fournisseur

Configuration des paramètres pour le fournisseur de messagerie.
