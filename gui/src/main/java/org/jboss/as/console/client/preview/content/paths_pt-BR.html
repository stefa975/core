Caminhos

O caminho do sistema de arquivos nomeado, mas sem uma solicitação para especificar o caminho atual. Se nenhum caminho de fato for especificado, ele atuará como um espaço reservado no modelo (ex.: no nível do domínio) até que uma definição de caminho totalmente especificada seja aplicada no nível mais inferior (ex.: no nível do host, onde os endereços disponíveis são conhecidos.)
