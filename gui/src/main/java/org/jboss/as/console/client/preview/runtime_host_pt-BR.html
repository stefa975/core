Configuração de host

Um controlador de host é um host físico ou virtual que interage com o controlador de domínio central para configurar e gerenciar as instâncias do servidor de aplicativos em execução no host.

Tarefas de configuração comuns

  • Configure as interfaces de rede, as definições de JVM e outras propriedade para este host.
  • Crie e gerencie instâncias de servidor para este host.
  • Visualize arquivos de log do servidor.
