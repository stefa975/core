Grupos de servidores

Um grupo de servidores é um conjunto de instâncias que serão configuradas e gerenciadas como uma. Em um domínio gerenciado, cada instância do servidor de aplicativos será um membro de um grupo de servidores, mesmo quando houver apenas um único servidor em um grupo de servidores.

O controlador de domínio e os controladores de host são responsáveis por garantir que todos os servidores em um grupo de servidores tenham uma configuração consistente. Todas as instâncias de servidor usam o mesmo perfil e tem o mesmo conteúdo de implantação.
