Grupo de servidores

Un grupo de servidores es un conjunto de instancias de servidor que se configuran y se administran como una sola. En un dominio administrado, todas las instancias de servidor son miembros de un grupo de servidores. Todas las instancias de servidor dentro de un grupo de servidores usan el mismo perfil y tienen el mismo contenido de implementación.

Tareas de configuración comunes

  • Agregar nuevos grupos de servidores.
  • Cree y asigne servidores a grupos de servidores.
  • Inicie, detenga y recargue servidores.
  • Consulte los archivos de registro del servidor.
