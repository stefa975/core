Grupos de servidores

Um grupo de servidores é um conjunto de servidores que serão gerenciados e configurados como um. Em um domínio gerenciado, cada instância do servidor de aplicativos é um membro de um grupo de servidores (mesmo que o grupo tenha apenas um único servidor, este ainda será membro de um grupo). Os Controladores de domínio e os Controladores de host são responsáveis por garantir que todos os servidores em um grupo de servidores tenham uma configuração consistente. Todos eles devem ser configurados com o mesmo perfil e ter o mesmo conteúdo de implantação.
