Zugriffssteuerung

Die Zugriffssteuerungsdefinitionen erteilen oder verhindern Zugriff auf Verwaltungsvorgänge.

Allgemeine Konfigurationsaufgaben

  • Benutzer und Gruppen hinzufügen.
  • Benutzern und Gruppen Rollen zuweisen.
  • Benutzerdefinierte Rollen für spezielle Hosts oder Servergruppen definieren.
