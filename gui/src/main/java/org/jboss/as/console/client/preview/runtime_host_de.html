Hostkonfiguration

Ein Host Controller ist ein physischer oder virtueller Host, der mit dem zentralen Domain Controller interagiert, um die Applikationsserverinstanzen, die auf dem jeweiligen Host ausgeführt werden, zu konfigurieren und zu verwalten.

Allgemeine Konfigurationsaufgaben

  • Netzwerk-Interfaces, JVM-Einstellungen und andere Eigenschaften für diesen Host konfigurieren.
  • Serverinstanzen für diesen Host erstellen und verwalten.
  • Serverprotokolldateien anzeigen.
