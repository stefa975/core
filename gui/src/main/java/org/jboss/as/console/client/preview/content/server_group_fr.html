Groupes de serveurs

Un groupe de serveurs est un regroupement d'instances de serveurs qui sont gérés et configurés en un. Dans un domaine géré, chaque instance de serveur d'applications appartient à un groupe de serveurs. Même si le groupe n'est associé qu'à un seul serveur, ce dernier appartient quand même à un groupe. Il incombe au contrôleur de domaine et aux contrôleurs d'hôtes de s'assurer que tous les serveurs d'un groupe disposent d'une configuration homogène. Tous doivent partager le même profil et le même contenu de déploiement.
