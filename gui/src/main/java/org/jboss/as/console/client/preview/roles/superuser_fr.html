Superutilisateur

Le rôle de superutilisateur n'a aucune restriction. Il permet d'accéder à toutes les ressources et opérations du serveur, y compris aux systèmes de journalisation d'audit et aux données sensibles. Ce rôle est équivalent à celui d'administrateur des versions antérieures. Si RBAC est désactivé, tous les utilisateurs de gestion disposent de permissions équivalentes à celles du rôle de superutilisateur.
