Interfaces

Ein Interface ist ein logischer Name für Netzwerkschnittstellen, IP-Adressen oder Hostnamen, an die Sockets gebunden werden können. Andere Abschnitte der Konfiguration können dann mittels deren logischem Namen auf diese Sockets verweisen, anstatt die vollständigen Details anzugeben (die auf verschiedenen Rechnern variieren können). Eine Interface-Konfiguration beinhaltet auch Informationen über die zum Auflösen der physischen Adresse zu verwendenden Kriterien.
