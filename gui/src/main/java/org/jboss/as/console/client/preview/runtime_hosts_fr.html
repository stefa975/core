Hôtes

Un domaine géré est composé de contrôleurs d'hôte, qui sont des hôtes physiques et virtuels qui interagissent avec le contrôleur de domaine central pour gérer les instances de serveurs d'applications exécutées sur son hôte. Un contrôleur d'hôte est configuré pour agir comme contrôleur de domaine central. Chaque contrôleur d'hôte peut avoir plusieurs groupes de serveurs.
