Groupe de serveurs

Un groupe de serveurs est un regroupement d'instances de serveurs qui sont gérés et configurés en un. Dans un domaine géré, chaque instance de serveur appartient à un groupe de serveurs. Toutes les instances de serveur d'un groupe partagent le même profil et le même contenu de déploiement.

Tâches de configuration courantes

  • Ajouter de nouveaux groupes de serveurs.
  • Créer et attribuer des serveurs aux groupes.
  • Démarrer, arrêter et recharger des serveurs.
  • Afficher les fichiers de journalisation des serveurs.
