Grupo de servidores

Um grupo de servidores é um conjunto de instâncias de servidor que serão configuradas e gerenciadas como uma. Em um domínio gerenciado, todas as instâncias de servidor são membros de um grupo de servidores. Todas as instâncias de servidor em um grupo de servidores usam o mesmo perfil e têm o mesmo conteúdo de implantação.

Tarefas de configuração comuns

  • Adicione um novo grupo de servidores.
  • Crie e atribua servidores a grupos de servidores.
  • Inicie, pare e recarregue servidores.
  • Visualize arquivos de log do servidor.
