Tiempo de ejecución

Administre hosts, grupos de servidores y servidores. Consulte y monitoree servicios de tiempo de ejecución, como archivos de registro, métricas JVM y datos del tiempo de ejecución específicos del subsistema.

Tareas de configuración comunes

  • Cree y configure grupos de servidores.
  • Cree y asigne servidores a grupos de servidores.
  • Consulte y monitoree métricas.
  • Inspeccione el estado del servidor.
  • Busque nombres de servicios de tiempo de ejecución relacionados con la aplicación.
