Interfaces

Uma interface é um nome lógico de uma interface de rede, um endereço IP ou um nome de host ao qual os soquetes podem se associar. Outras áreas da configuração podem, então, referenciar a interface por seu nome lógico em vez de fornecerem detalhes completos, que podem variar em diferentes máquinas. Uma configuração de interface também especifica critérios para a resolução do endereço físico real que deve ser usado.
