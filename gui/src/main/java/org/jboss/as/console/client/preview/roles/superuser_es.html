Superusuario

El rol del superusuario no tiene restricciones y tiene acceso completo a todos los recursos y las operaciones del servidor, incluidos datos confidenciales y el sistema de registro de auditorías. Este rol es equivalente al de los usuarios administradores de versiones anteriores. Si RBAC está deshabilitado, todos los usuarios de administración tienen permisos equivalentes a los del rol del superusuario.
