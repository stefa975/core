Runtime

Affichage et surveillance des services runtime, comme les fichiers de journalisation, les métriques de JVM et les données d'exécution spécifiques aux sous-systèmes.

Tâches de configuration courantes

  • Affichage et surveillance des métriques.
  • Inspection de l'état des serveurs.
  • Recherche de noms de services runtime liés aux applications.
