Administrador

A função Administrador tem acesso irrestrito a todos os recursos e a todas as operações no servidor, com exceção do sistema de registro em log de auditoria. A função Administrador tem acesso a dados e operações confidenciais. Esta função também pode configurar o sistema de controle de acesso. A função Administrador só é necessária para lidar com dados confidenciais ou configurar usuários e funções. Os Administradores não podem alterar suas permissões para Auditor ou Superusuário.
