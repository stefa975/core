Caminhos

Um caminho é um nome lógico para um caminho do sistema de arquivos. Outras áreas da configuração podem, então, referenciar o caminho usando o nome lógico em vez do caminho completo, que pode variar em máquinas diferentes. Há vários caminhos predefinidos em uso pela configuração padrão, como o caminho jboss.server.log.dir, que define o diretório do log do servidor.
