Superusuário

A função Superusuário não tem restrições e tem acesso completo a todos os recursos e todas as operações do servidor, incluindo o sistema de registro em log de auditoria e dados confidenciais. Essa função é equivalente aos usuários administradores de versões anteriores. Se o RBAC for desativado, todos os usuários de gerenciamento terão permissões equivalentes às da função Superusuário.
