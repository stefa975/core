Servergruppen

Eine Servergruppe ist eine Reihe aus Serverinstanzen, die als eine Einheit verwaltet und konfiguriert werden. In einer verwalteten Domain gehört jede Applikationsserverinstanz zu einer Servergruppe. (Selbst wenn die Gruppe nur einen Server enthält, ist der Server dennoch Mitglied einer Gruppe.) Der Domain Controller und die Host Controller müssen sicherstellen, dass alle Server in einer Servergruppe eine konsistente Konfiguration aufweisen. Sie sollten alle mit demselben Profil konfiguriert sein und über denselben Deployment-Inhalt verfügen.
