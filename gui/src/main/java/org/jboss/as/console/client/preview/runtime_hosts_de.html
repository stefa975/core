Hosts

Eine verwaltete Domain besteht aus Host Controllern. Das sind physische oder virtuelle Hosts, die mit dem zentralen Domain Controller interagieren, um die Applikationsserverinstanzen, die auf dem jeweiligen Host ausgeführt werden, zu verwalten. Ein Host Controller wird als zentraler Domain Controller konfiguriert. Jeder Host Controller kann mehrere Servergruppen aufweisen.
