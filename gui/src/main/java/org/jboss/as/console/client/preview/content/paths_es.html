Rutas

Una ruta nombrada del sistema de archivos, pero sin ningún requerimiento para especificar la ruta real. Si no se especifica una ruta, actúa como un comodín en el modelo (por ejemplo, a nivel del dominio) hasta que se aplica una definición de ruta completamente especificada en un nivel más bajo (por ejemplo, a nivel de host, donde se conocen las direcciones disponibles).
