Configuración de hosts

Un controlador de hosts es un host físico o virtual que interactúa con el controlador de dominios central para configurar y administrar las instancias del servidor de aplicaciones que se ejecutan en el host.

Tareas de configuración comunes

  • Configure interfaces de red, ajustes de JVM y otras propiedades para este host.
  • Cree y administre instancias del servidor para este host.
  • Consulte los archivos de registro del servidor.
