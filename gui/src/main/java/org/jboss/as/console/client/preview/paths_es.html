Rutas

Una ruta es un nombre lógico para una ruta del sistema de archivos. Otras áreas de la configuración pueden hacer referencia a la ruta que usa el nombre lógico en lugar de la ruta completa, que puede variar según la máquina. Hay varias rutas predefinidas utilizadas por la configuración predeterminada, como, por ejemplo, la ruta jboss.server.log.dir que define el directorio para el registro de servidores.
