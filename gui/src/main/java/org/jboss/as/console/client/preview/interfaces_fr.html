Interfaces

Une interface est un nom logique d'interface de réseau/d'adresse IP/de nom d'hôte auquel les sockets peuvent être liés. D'autres sections de la configuration peuvent alors référencer l'interface par leur nom logique, plutôt que d'avoir à inclure tous les détails de l'interface (ce qui peut varier selon les machines). Une configuration de l'interface indique également les critères à utiliser pour résoudre l'adresse physique réelle à utiliser.
