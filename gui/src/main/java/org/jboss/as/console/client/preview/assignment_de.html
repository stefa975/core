Zuweisung

Ein Benutzer gilt als einer Rolle zugewiesen, wenn Folgendes zutrifft:

  1. Der Benutzer ist:
    • als Benutzer aufgelistet, der in die Rolle aufgenommen werden soll, oder
    • ein Mitglied bei einer Gruppe, die in die Rolle aufgenommen werden soll.
  2. Der Benutzer ist nicht:
    • als Benutzer aufgelistet, der aus der Rolle ausgeschlossen werden soll, oder
    • ein Mitglied bei einer Gruppe, die aus der Rolle ausgeschlossen werden soll.

Ausschlüsse haben Vorrang vor Aufnahmen.
