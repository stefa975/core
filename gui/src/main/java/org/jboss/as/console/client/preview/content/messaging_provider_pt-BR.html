Provedor de mensagens

Um provedor de mensagens é uma instância do servidor ActiveMQ.

Filas/tópicos

Crie filas e tópicos usados por este provedor de mensagens.

Conexões

Gerencie aceitadores, que definem como é possível fazer conexões com o provedor de mensagens.

Crie bridges, que consomem mensagens de uma fila de origem e as encaminham para um endereço de destino, em geral, em um servidor diferente.

Clustering

Defina grupos de difusão e de descoberta, que controlam como os conectores são manipulados e distribuídos por este provedor.

Gerencie conexões de cluster para balancear a carga entre os nós de um cluster.

Configurações do provedor

Configure as definições deste provedor de mensagens.
