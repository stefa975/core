Betreuer

Die Betreuer-Rolle kann den Laufzeitstatus und alle Konfigurationen, mit Ausnahme von sensiblen Daten und Vorgängen, anzeigen und ändern. Die Betreuer-Rolle ist die allgemeine Rolle ohne Zugriff auf sensible Daten und Vorgänge. Mit dieser Rolle erhalten Benutzer fast vollständigen Zugriff auf die Serveradministration, ohne dass sie auf Passwörter und andere sensible Daten zugreifen können.
