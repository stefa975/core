Interfaces

Una interfaz es un nombre lógico para una interfaz de red, dirección IP o nombre de host al cual se pueden enlazar los sockets. Otras secciones de la configuración pueden hacer referencia a esas interfaces por medio de su nombre lógico en lugar de tener que proveer todos los detalles de la interfaz, que pueden variar según la máquina. Una configuración de interfaz también especifica los criterios a utilizar para resolver la dirección física que se va a utilizar.
