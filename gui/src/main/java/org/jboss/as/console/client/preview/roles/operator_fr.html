Opérateur

Le rôle d'opérateur étend les attributions du rôle de surveillant en y ajoutant la capacité de modifier l'état d'exécution du serveur. Cela signifie que les opérateurs peuvent recharger et arrêter le serveur, mais aussi interrompre et reprendre les destinations JMS. Le rôle d'opérateur est idéal pour les utilisateurs responsables des hôtes physiques ou virtuels du serveur d'applications, qui peuvent ainsi s'assurer que les serveurs peuvent être arrêtés et redémarrés correctement si besoin. Les opérateurs ne peuvent ni modifier une configuration serveur ni accéder à des opérations ou données sensibles.
