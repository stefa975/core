Chemins d'accès

Un chemin est un nom logique de chemin d'accès de système de fichiers. D'autres sections de configuration peuvent alors référencer ces chemins d'accès par leur nom logique, plutôt que d'avoir à inclure tous les détails du chemin (qui peut varier selon les machines). Il existe un certain nombre de chemins prédéfinis qui sont utilisés par la configuration par défaut comme jboss.server.log.dir qui pointe vers le répertoire "log" du serveur.
